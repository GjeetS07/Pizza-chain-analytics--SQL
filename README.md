Pizza Sales & Delivery Analysis (SQL Project)

Project Overview
This project focuses on analyzing pizza sales and delivery performance using SQL. The goal is to extract meaningful business insights that can help improve operational efficiency, customer satisfaction, and revenue generation.

Tools & Technologies

* SQL (MySQL)
* Data Analysis
* Business Insights

Dataset Description
The dataset contains detailed information about pizza orders including:
* Order details (date, time, location)
* Delivery performance (time, delay, distance)
* Customer preferences (pizza type, toppings)
* External factors (traffic, peak hours, weekends)

Key Business Questions Solved
Sales Insights
* Most ordered pizza type
* Orders by month
* Customer topping preferences

Delivery Performance
* Average delay by traffic level
* Delivery efficiency (time per km)
* Most delayed orders

Operational Insights
* Peak order hours
* Weekend vs weekday demand
* Impact of traffic on delivery

Payment Analysis
* Card vs Cash usage distribution

Key Insights
* Identified peak hours with highest order volume
* High traffic significantly increases delivery delays
* Certain locations generate more orders with faster delivery
* Customer preference trends in pizza types and toppings
